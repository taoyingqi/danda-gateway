## 项目结构
* /config _微信、代理、数据库、日志配置_
* /controllers _控制层_
* /logs _日志文件_
* /models _sequelize model 定义_
* /routes _express route_
* /app.js _入口文件_
* /package.json
* /start _启动脚本_

# deployed on linux

2. install
* node-v8.10
* expose PORT=10020
* chmod +x start
